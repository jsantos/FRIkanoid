Asounding
=========

Version 1.0 - 1 Sep 2009

by Michel Fortin
<http://michelf.com/>


Introduction
------------

Asounding is an audio engine for iPhone OS specially built for games. The engine provides a simple model for loading an playing sound effects and background music while correctly handling system events and iPod music playback in the background.


Requirements
------------

Asounding requires iPhone OS 2.2.1 or later.
